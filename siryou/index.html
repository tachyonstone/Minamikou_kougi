CSS

レイアウトや装飾など見た目を整えるための言語．

CSSの基本的な書き方

どれの {何を: どうするか}

3通りの書き方

  • タグのstyle属性に直接記述
  • style要素内に記述
  • 外部ファイルに記述
直接記述． 

<p style="color:blue;">
style要素内に記述 

<style type="text/css">
p {
  color: blue;
}
</style>
<p>hello.</p>
CSSファイルを外部から読み込む． 

<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="#">
    <title>TEST</title>
  </head>

  <body>
    <h1>test</h1>

    <p>This is a test script. </p>

  </body>

</html>
